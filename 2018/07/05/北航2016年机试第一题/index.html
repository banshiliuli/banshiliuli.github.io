逆序数

题目描述

给定一个数n，将这个数的各位顺序颠倒，称为逆序数m。 例如1234的逆序数是4321。

输入

输入一个数n, n开头无多余的0（0 < n > 1000000000）

输出

如果m是n的k倍（k为整数），那么输出n*k=m。 如果m不是n的整数倍，那么输出n和n的逆序数。
